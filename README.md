# DNAEncodedLibraryEngine

DNA-encoded chemical library (DEL) analysis for hit identification, affinity selection, and chemical space exploration.

## Features

- DEL hit identification from selection enrichment data
- Scaffold diversity analysis using Shannon entropy
- Affinity selection simulation across 1M compounds
- True positive rate and false discovery rate evaluation
- Chemical space coverage and clustering analysis

## Results

1M compounds (100K sampled); 319 hits (0.32%); TPR=100%; Scaffold diversity H=4.60

## Usage

```bash
pip install numpy scipy matplotlib
python dna_encoded_library_engine.py
```
